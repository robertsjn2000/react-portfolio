# react-portfolio

## Description

This is a portfolio made using REACT. It allows me to display my work for potential employeers.


## Installation

Once the repo is cloned down ensure that all dependencies are installed and in the CLI run "npm run start" to spin up the server.

## Usage

Simply run the deoplyed link.

https://robertsjn2000.github.io/react-portfolio/
